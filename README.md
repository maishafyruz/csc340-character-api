# csc340-character-api

This project is a restful CRUD API built with Spring Boot, Spring Data JPA and PostgreSQL (Neon).  
It manages **character records** for a character website, including create, read, update, delete, category filtering and name search.



## Tech Stack
- Java
- Spring Boot (Spring Web, Spring Data JPA)
- PostgreSQL (Neon.tech)
